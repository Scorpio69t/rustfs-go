# Change: 添加高级对象/桶功能与文件直传助手

## Why
- 客户端需要补齐 S3 兼容的高级能力（预签名、标签、桶策略、生命周期、服务端加密），当前 SDK 尚未覆盖。
- 直接基于文件路径上传/下载的便捷方法缺失，用户需要手工处理文件流，易错且繁琐。

## What Changes
- 增加对象预签名 GET/PUT 能力，支持自定义过期与可选响应头。
- 提供对象标签的创建、读取、删除接口。
- 提供桶策略的设置、获取、删除接口。
- 提供桶生命周期配置的设置、获取、删除接口。
- 在上传/下载相关接口中支持服务端加密（SSE-S3、SSE-C）选项。
- 为对象操作新增基于文件路径的 fput/fget 便捷方法，自动处理文件打开、校验和元数据推断。

## Impact
- 影响的规范：core-api
- 影响的代码：object 上传/下载/预签名模块，bucket 策略与生命周期模块，加密和选项定义，客户端入口与便捷方法
